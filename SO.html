Windows 1.0 – 1985

O Windows 1.0 foi o primeiro sistema operacional gráfico desenvolvido pela Microsoft. Lançado em 1985, ele ofereceu uma interface com janelas sobreponíveis, mas ainda dependia do MS-DOS. Não foi um grande sucesso, mas iniciou a jornada dos sistemas gráficos da Microsoft.

Windows 2.0 – 1987

O Windows 2.0 foi uma atualização significativa, introduzindo suporte para janelas sobreponíveis e novos atalhos de teclado. Embora ainda fosse dependente do MS-DOS, ele trouxe maior capacidade de multitarefa e foi mais estável.

Windows 3.0 – 1990

Com o Windows 3.0, a Microsoft introduziu um sistema operacional muito mais eficiente, com suporte a mais cores, melhores gráficos e novos recursos de multitarefa. Foi a primeira versão de sucesso comercial do Windows.

Windows 3.1 – 1992

O Windows 3.1 foi uma versão aprimorada do Windows 3.0, oferecendo maior estabilidade e suporte para fontes TrueType, que tornaram as fontes mais legíveis. Também foi o primeiro Windows a ganhar popularidade significativa entre usuários domésticos.

Windows 95 – 1995

O Windows 95 foi um marco na história do sistema operacional. Com a introdução do menu Iniciar, barra de tarefas e plug-and-play, o Windows 95 marcou a transição para um sistema operacional mais moderno e fácil de usar.

Windows 98 – 1998

O Windows 98 trouxe melhorias de desempenho, melhor suporte a hardware e estabilidade. A introdução do Windows Update foi uma das principais novidades. Foi um dos sistemas mais populares antes do lançamento do Windows XP.

Windows ME (Millennium Edition) – 2000

O Windows ME foi uma tentativa de modernizar o Windows 98, mas teve uma recepção negativa devido à sua instabilidade e falhas técnicas. Apesar disso, trouxe novos recursos multimídia e um foco maior em conectividade com a Internet.

Windows 2000 – 2000

O Windows 2000 foi projetado tanto para o uso corporativo quanto doméstico, com maior estabilidade e suporte a sistemas de rede. Foi muito popular entre profissionais de TI devido à sua robustez.

Windows XP – 2001

O Windows XP foi um dos sistemas operacionais mais bem-sucedidos da Microsoft. Ele combinava a estabilidade do Windows 2000 com a interface amigável e recursos multimídia mais modernos. Sua longevidade e popularidade são lendárias.

Windows Vista – 2007

O Windows Vista foi lançado com um visual muito moderno para a época, com uma nova interface Aero. No entanto, a performance e a compatibilidade de hardware foram amplamente criticadas, o que afetou sua popularidade.

Windows 7 – 2009

O Windows 7 foi uma resposta às falhas do Vista, com melhorias significativas de desempenho, estabilidade e interface. Tornou-se um dos sistemas operacionais mais amados e duradouros da Microsoft.

Windows 8 – 2012

O Windows 8 introduziu a interface Metro e uma abordagem mais voltada para dispositivos touchscreen. Embora tenha sido inovador, a mudança radical no design causou confusão entre os usuários de desktop tradicionais.

Windows 8.1 – 2013

O Windows 8.1 foi uma atualização do Windows 8, trazendo de volta o menu Iniciar e melhorias na interface. A Microsoft tentou corrigir as falhas do Windows 8 com essa versão, mas a transição ainda foi desafiadora para muitos usuários.

Windows 10 – 2015

O Windows 10 combinou o melhor de versões anteriores, como o menu Iniciar, com a flexibilidade para dispositivos touchscreen. Foi bem recebido pelos usuários, com atualizações constantes e melhorias em segurança.

Windows 11 – 2021

O Windows 11 traz um design mais moderno, com bordas arredondadas e a centralização do menu Iniciar. Foi otimizado para as necessidades de novos dispositivos e melhora a experiência de uso em dispositivos híbridos e de tela sensível ao toque.

Ubuntu

Ubuntu é uma das distribuições Linux mais populares, baseada no Debian. Lançada pela primeira vez em 2004, seu objetivo é oferecer uma interface amigável e fácil de usar, com foco em ser uma opção acessível para iniciantes. O Ubuntu é amplamente usado tanto em desktops quanto em servidores e é conhecido pela estabilidade e pela vasta documentação.

Debian

Debian é uma das distribuições mais antigas e respeitadas, conhecida por sua estabilidade e segurança. Criado por Ian Murdock em 1993, o Debian serve de base para muitas outras distribuições, como o Ubuntu. Debian é utilizado principalmente em servidores, mas também possui versões desktop. É ideal para quem busca um sistema sólido e livre de distrações.

Fedora

Fedora é uma distribuição patrocinada pela Red Hat e focada em trazer as tecnologias mais recentes do mundo open-source. Seu lançamento acontece a cada 6 meses, com a intenção de ser uma plataforma de inovação, sendo mais voltada para desenvolvedores e entusiastas de tecnologia. O Fedora oferece uma experiência "pura" do GNOME como ambiente de desktop padrão.

Arch Linux

Arch Linux é uma distribuição minimalista e altamente configurável. Lançada em 2002, ela segue o princípio KISS (Keep It Simple, Stupid), oferecendo aos usuários um controle completo sobre o sistema. Arch é indicada para usuários avançados, já que a instalação e configuração exigem mais conhecimento técnico. Ela utiliza o modelo rolling release, o que significa que o sistema é constantemente atualizado.

Linux Mint

Linux Mint é uma distribuição baseada no Ubuntu, focada na simplicidade e na facilidade de uso. Criada para ser uma opção amigável para iniciantes, ela oferece uma interface gráfica limpa e agradável. Mint vem com uma variedade de software pré-instalado, o que facilita a vida dos usuários que não querem configurar tudo do zero. Ela é muito popular entre usuários de desktop.

Manjaro

Manjaro é uma distribuição baseada no Arch Linux, mas com um foco em tornar o Arch acessível para iniciantes. Lançada em 2011, ela vem com uma interface mais amigável e uma instalação simplificada. Manjaro mantém as vantagens do Arch, como o modelo rolling release, mas oferece uma experiência mais fácil de usar, com a possibilidade de escolher entre diferentes ambientes gráficos como XFCE, KDE e GNOME.

CentOS

CentOS é uma distribuição Linux baseada no Red Hat Enterprise Linux (RHEL), projetada para ser uma versão gratuita e de código aberto do RHEL. CentOS é muito usado em servidores, especialmente em ambientes corporativos que precisam de uma plataforma robusta e confiável. Embora o CentOS tenha sido descontinuado como um projeto independente, ele ainda é usado em algumas versões do CentOS Stream, que serve como um projeto intermediário para RHEL.

ArchBang

ArchBang é uma distribuição baseada no Arch Linux, mas com uma instalação mais simples e pré-configurada, tornando-a uma boa escolha para quem quer a filosofia do Arch, mas não deseja configurar tudo do zero. Ela oferece uma interface gráfica minimalista, utilizando o Openbox como gerenciador de janelas.

Mac OS X 10.0 Cheetah – 2001

Primeira versão oficial do Mac OS X, lançada em março de 2001. Introduziu a nova interface Aqua, baseada em Unix, com o kernel Darwin. Trouxe estabilidade e segurança em comparação ao Mac OS 9, mas tinha desempenho lento e poucas aplicações compatíveis no início.

Mac OS X 10.1 Puma – 2001

Lançada em setembro de 2001, focou em melhorias de desempenho e usabilidade. Tornou o sistema mais rápido e estável, adicionando suporte a impressoras e melhorias no Finder. Também trouxe suporte a DVDs e melhor integração com hardware.

Mac OS X 10.2 Jaguar – 2002

Lançada em agosto de 2002, Jaguar foi a primeira versão a ter grande aceitação comercial. Melhorou a velocidade, adicionou suporte a Bluetooth, melhores drivers de impressora, e trouxe o Spotlight, sistema de busca avançada.

Mac OS X 10.3 Panther – 2003

Lançada em outubro de 2003, Panther trouxe grandes melhorias visuais e funcionais, como o Exposé (para organizar janelas), o Safari atualizado, suporte a PDF nativo, e o iChat AV com vídeo.

Mac OS X 10.4 Tiger – 2005

Disponível em abril de 2005, Tiger introduziu o Spotlight em nível de sistema, Dashboard com widgets, melhorias no Safari, suporte ao 64 bits, e um novo framework de desenvolvimento (Core Animation).

Mac OS X 10.5 Leopard – 2007

Lançada em outubro de 2007, Leopard foi uma atualização significativa que trouxe a Time Machine para backups automáticos, suporte a 64 bits, Spaces para múltiplas áreas de trabalho, e um novo Dock 3D.

Mac OS X 10.6 Snow Leopard – 2009

Lançada em agosto de 2009, focou em otimização e desempenho, reduzindo o tamanho do sistema e melhorando a estabilidade. Adicionou suporte aprimorado a 64 bits, e melhorias no Finder e no QuickTime.

Mac OS X 10.7 Lion – 2011

Disponível em julho de 2011, Lion trouxe muitas funcionalidades do iOS para o desktop, como Launchpad, suporte a gestos multitouch, Auto Save e versões de documentos. Também introduziu o Mac App Store.

OS X 10.8 Mountain Lion – 2012

Lançada em julho de 2012, focou em maior integração com o iCloud, trouxe o Centro de Notificações, mensagens via iMessage no Mac, Dictation e melhorias no Safari e Mail.

OS X 10.9 Mavericks – 2013

Lançada em outubro de 2013, Mavericks trouxe melhorias na eficiência energética, Finder com abas, tags para organização de arquivos, e melhor suporte para múltiplos monitores. Foi a primeira versão gratuita do OS X.

OS X 10.10 Yosemite – 2014

Disponível em outubro de 2014, Yosemite reformulou a interface com design mais plano e moderno, similar ao iOS 7. Introduziu o recurso Continuidade para integração entre Mac e iPhone, além do iCloud Drive.

OS X 10.11 El Capitan – 2015

Lançada em setembro de 2015, focou em desempenho e estabilidade, melhorando a experiência com janelas (Split View), aprimoramentos no Spotlight, e melhorias no Mail e Safari.

macOS 10.12 Sierra – 2016

A primeira versão a usar o nome “macOS”. Lançada em setembro de 2016, trouxe a Siri para o Mac, suporte ao Apple Pay na web, melhorias no iCloud Drive e no Auto Unlock com Apple Watch.

macOS 10.13 High Sierra – 2017

Disponível em setembro de 2017, focou em melhorias técnicas, incluindo o novo sistema de arquivos APFS, suporte a HEVC para vídeos, Metal 2 para gráficos e melhorias em Safari e Mail.

macOS 10.14 Mojave – 2018

Lançada em setembro de 2018, Mojave trouxe o modo escuro, área de trabalho dinâmica, pilhas para organização automática de arquivos, e melhorias na privacidade e segurança.

macOS 10.15 Catalina – 2019

Disponível em outubro de 2019, Catalina substituiu o iTunes por três apps separados (Música, Podcasts, TV), introduziu o Sidecar para usar o iPad como segunda tela, e trouxe o Catalyst para portar apps iPad para Mac.

macOS 11 Big Sur – 2020

Lançada em novembro de 2020, trouxe o maior redesign visual em anos, com uma interface mais leve e moderna. Foi a primeira versão a suportar Macs com chips Apple Silicon (M1).

macOS 12 Monterey – 2021

Disponível em outubro de 2021, Monterey focou na continuidade e colaboração, com o SharePlay, melhorias no FaceTime, Atalhos para Mac, e Universal Control para usar mouse e teclado em vários dispositivos.

macOS 13 Ventura – 2022

Lançada em outubro de 2022, Ventura trouxe Stage Manager para organização automática das janelas, melhorias no Spotlight, novas funcionalidades no Mail e Mensagens, e suporte ao Handoff para apps de vídeo.

macOS 14 Sonoma – 2023

Anunciada e lançada em 2023, Sonoma foca em melhorias na performance, widgets interativos na área de trabalho, melhorias no Safari com perfis e suporte a jogos via Metal 3.

macOS 15 Sequoia – 2024

O macOS Sequoia foi anunciado na WWDC em 10 de junho de 2024, é o sucessor do macOS Sonoma, e a vigésima primeira e atual versão do macOS. É a primeira versão do sistema da Apple disponibilizado com a tecnologia IA (Apple Intelligence)
